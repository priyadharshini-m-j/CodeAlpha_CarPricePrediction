# CodeAlpha_CarPricePrediction

Data Science Internship Projects
This repository contains the projects completed as part of my Data Science Internship.
Tools Used
Python
Pandas
NumPy
Matplotlib & Seaborn
Scikit-learn
Jupyter Notebook
Claude AI
Projects
1. Iris Flower Classification
Trained a machine learning model to classify Iris flowers (Setosa, Versicolor, Virginica) based on sepal and petal measurements. Compared Logistic Regression, KNN, SVM, and Random Forest models. Best model achieved 100% accuracy.
2. Unemployment Analysis with Python
Analyzed unemployment rate data across Indian states (2019-2020), studied the impact of Covid-19 on unemployment trends, and identified seasonal and regional patterns.
3. Car Price Prediction with Machine Learning
Built a regression model to predict used car selling prices based on features like present price, car age, kilometers driven, and fuel type. Achieved 96% R² accuracy using Random Forest.
4. Sales Prediction using Python
Predicted product sales based on advertising spend across TV, Radio, and Newspaper channels, and identified which channel drives the most sales. Achieved 98% R² accuracy using Random Forest.
